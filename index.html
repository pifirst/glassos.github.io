* Обновления GLASS OSза ноябрь 2019 устанавливается автоматически с помощью GLASS OS Update. После завершения загрузки мы сообщим вам, чтобы выбрать правильное время для завершения установки. Если для вашей версии GLASS OS наступил срок прекращения поддержки, воспользуйтесь помощником по обновлению, чтобы обновить сейчас.

** Для поддерживаемого срока службы устройства; новые обновления функций автоматически устанавливаются на устройствах, которые имеют достаточно свободного места на диске, через службу GLASS OS Update. По возможности GLASS OS автоматически освобождает место, когда его не хватает, и помогает вам очистить еще больше пространства, если автоматической очистки недостаточно. Устройства, не имеющие нужного количества свободного места на диске, будут получать только автоматические обновления безопасности, позволяет место. Обновления функций будут доступны для загрузки пользователем, но не будут автоматически установлены через службу GLASS OS Update. Может взиматься плата за услуги интернет-провайдера. Впоследствии для обновлений могут применяться значительные дополнительные требования к хранилищу и / или аппаратного обеспечения. Более подробную информацию см. здесь.
